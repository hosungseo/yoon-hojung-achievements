## 사이트 소개

대한민국 정부조직 대개편, 경찰국 폐지, AI 정부24+, 민생회복 소비쿠폰, 전남광주통합특별시 출범 등
**행정안전부가 추진한 핵심 정책 성과**를 7개 카테고리로 정리한 공식 홍보용 웹사이트입니다.

### 7개 카테고리

1. **역사적 정부혁신** — 정부조직 대개편, 경찰국 폐지, 중대범죄수사청법 제정
2. **민생회복** — 민생회복 소비쿠폰, 지역사랑상품권, 지방세제 개편
3. **지방의 미래** — 전남광주통합특별시, 재정분권, 지방소멸대응기금 개편
4. **AI 민주정부** — AI 국민비서, AI 정부24+, 공공문서 혁신
5. **국민 안전** — 보이스피싱 통합대응, 마약범죄 대응, 재난안전 종사자 처우 개선
6. **민주주의의 회복** — 빛의 혁명 기여자 포상, 진실·화해위원회, 사회연대경제기본법
7. **새로운 행안부** — 국민 삶 플러스 비전, 여성 1급 임명, 대국민 업무보고 공개

### 페이지 구성

- **랜딩(1페이지 스크롤)**: 히어로 → 7개 카테고리 → 장관 메시지 → 제도화 타임라인
- **카테고리별 상세**: 각 카테고리마다 세부 성과, 근거 자료, 제도화 경과를 담은 전용 페이지
- **좌측 고정 내비게이션**: 데스크톱에서 섹션 간 빠른 이동
